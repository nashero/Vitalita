## Donor vs. Organization CTA Differentiation

### Detection Method: URL Parameter ⭐ RECOMMENDED

**Implementation:**
- `/how-we-simplify?audience=donor` → Donor CTAs
- `/how-we-simplify?audience=organization` → Organization CTAs
- Default: Organization (primary audience)

## Conversion Path Summary

### Path 1: Implementation Plan (Recommended)
```
Header: "Get Implementation Plan"
    ↓
Contact Form (Implementation Plan selected)
    ↓
Email: "Your 2-Week Implementation Plan" (auto-sent)
    ↓
Follow-up: "Ready to start? Book your onboarding call"
    ↓
Onboarding Call → Pilot Started
```

### Path 2: ROI Calculator
```
Hero: "See Your Time Savings"
    ↓
ROI Calculator Tool
    ↓
Results: "You'll save X hours/week"
    ↓
CTA: "Get Your Implementation Plan"
    ↓
Contact Form
```

### Path 3: Demo/Overview
```
Header: "Watch 2-Min Overview"
    ↓
Video Modal or /overview page
    ↓
Mid-video CTA: "See Full Demo" or "Get Implementation Plan"
    ↓
Contact Form or Demo Scheduling
```

### Path 4: Free Pilot (Direct)
```
Footer: "Start Free Pilot"
    ↓
Contact Form (Free Pilot selected)
    ↓
Email: "Welcome to Your Free Pilot"
    ↓
Onboarding: Week 1 Setup & Training
    ↓
Pilot Active (2 weeks)
    ↓
End of Pilot: "Ready to continue?"
```

## Design Specifications Summary

### Primary CTA (Large)
- Height: 56px
- Padding: 16px 32px
- Font: 16px, semibold
- Color: Red gradient (#FF6B6B to #E65A5A)
- Border Radius: 12px
- Shadow: 0 4px 16px rgba(255, 107, 107, 0.3)
- Hover: Scale 1.02, shadow increase, copy change

### Primary CTA (Medium)
- Height: 48px
- Padding: 12px 24px
- Font: 15px, semibold
- Same styling as large, smaller size

### Secondary CTA
- Height: 44px
- Padding: 10px 20px
- Font: 14px, semibold
- Color: White background, red border (2px)
- Hover: Border-red-500, bg-red-50

### Micro CTA (Text Link)
- Font: 14px, semibold
- Color: text-red-600
- Hover: underline, text-red-700
- Icon: Optional arrow (4px gap)
