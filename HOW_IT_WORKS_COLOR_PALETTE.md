# How It Works Page - Color Palette Documentation

## Official Color Palette

This document defines the complete color palette used on the "How It Works" page, matching the Home Page exactly.

## Primary Brand Colors

### Primary Red
- **Hex**: `#FF6B6B`
- **RGB**: `rgb(255, 107, 107)`
- **Usage**: 
  - Primary brand color
  - All accent elements
  - Icons, badges, buttons
  - Progress indicators
  - Data visualization
- **Hover State**: `#E65A5A` (RGB: `rgb(230, 90, 90)`)
- **Opacity Variants**:
  - `rgba(255, 107, 107, 0.08)` - Light backgrounds
  - `rgba(255, 107, 107, 0.1)` - Icon backgrounds
  - `rgba(255, 107, 107, 0.12)` - Step 2 background
  - `rgba(255, 107, 107, 0.16)` - Step 3 background
  - `rgba(255, 107, 107, 0.2)` - Step 4 background
  - `rgba(255, 107, 107, 0.2)` - Border opacity
  - `rgba(255, 107, 107, 0.3)` - Border opacity (Step 2)
  - `rgba(255, 107, 107, 0.4)` - Border opacity (Step 3)
  - `rgba(255, 107, 107, 0.5)` - Border opacity (Step 4)
  - `rgba(255, 107, 107, 0.3)` - Shadow opacity
  - `rgba(255, 107, 107, 0.4)` - Shadow opacity (hover)

### Primary Text
- **Hex**: `#111827`
- **RGB**: `rgb(17, 24, 39)`
- **Usage**:
  - All primary headings (H1, H2, H3)
  - Step titles
  - FAQ questions
  - CTA titles
  - Mockup card headers
  - Primary data values
- **Contrast**: 15.8:1 on white (AAA)

### Secondary Text
- **Hex**: `#6B7280`
- **RGB**: `rgb(107, 114, 128)`
- **Usage**:
  - Body text
  - Descriptions
  - Secondary information
  - Step descriptions
  - FAQ answers
  - CTA descriptions
- **Contrast**: 6.2:1 on white (AA)

## Usage Rules

### Do's
✅ Use `#FF6B6B` for all primary brand elements  
✅ Use `#111827` for all primary headings  
✅ Use `#6B7280` for body text and descriptions  
✅ Use `#F9FAFB` for main backgrounds  
✅ Use opacity variants for subtle backgrounds  
✅ Use status colors with text labels  
✅ Maintain consistent hover states  

### Don'ts
❌ Don't use different red shades (stick to `#FF6B6B`)  
❌ Don't use Tailwind default colors (use exact hex values)  
❌ Don't use color alone to convey information  
❌ Don't use low contrast combinations  
❌ Don't deviate from the Home Page color scheme

## WCAG 2.2 Compliance

### Contrast Ratios
- **Primary Text (`#111827`) on White**: 15.8:1 ✅ AAA
- **Secondary Text (`#6B7280`) on White**: 6.2:1 ✅ AA
- **Red Text (`#FF6B6B`) on White**: 3.8:1 ⚠️ (Large text only)
- **White Text on Red (`#FF6B6B`)**: 4.5:1 ✅ AA
- **Muted Text (`#9CA3AF`) on White**: 4.6:1 ✅ AA (Large text)

### Accessibility Notes
- All status indicators have text labels
- Information not conveyed by color alone
- Focus indicators meet contrast requirements
- All interactive elements have visible focus states
